\subsection{Floating bodies}

\begin{tabular}{@{}p{\the\MyLen}%
                @{}p{\linewidth-\the\MyLen}@{}}

The \textit{place} is a list valid placements for the body.  \texttt{t}=top,
\texttt{h}=here, \texttt{b}=bottom, \texttt{p}=separate page,  Captions and label markers should be within the environment.

\section{Tabular environments}

Remember to place a Tabular environment inside an table environment. Then it is easily possible to add a caption and the table will be added to the list of tables:

\verb!\begin{table}{!\textit{<pos>}\verb!}! \\
\verb!\begin{tabular}{|c|c|}!\\
\verb!\hline!\\
\verb!	11 & 12\\!\\
\verb!\hline!\\
\verb!	21 & 22\\!\\
\verb!\hline!\\
\verb!\end{tabular}!\\
\verb!\end{table}! \\

\section{Math mode}
Include in preamble:\\~\\
\verb!\usepackage{amsmath,amssymb,amsfonts,amsthm,mathtools}!\\~\\
For inline math, use \verb!\(...\)! or \verb!$...$!.
For displayed math, use \verb!\[...\]! or \verb!\begin{equation}!.
